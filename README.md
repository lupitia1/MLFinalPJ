## Repository contents
- data/  
  Folder containing the dataset.

- utils.jl  
  Auxiliary utilities: data splitting (holdOut, crossvalidation), normalization, one‑hot encoding, training helpers, metric calculations and confusion matrix helpers.

- <approach>.ipynb  
  Notebooks implementing each approach and experiments (ANN, SVM, Decision Tree, kNN, PCA, stacking). Each notebook runs multiple models and reports metrics and confusion matrices.
  - CrossValidation.ipynb
  - CrossValidation+PCA.ipynb
  - PCA.ipynb
  - AllDimensions.ipynb
